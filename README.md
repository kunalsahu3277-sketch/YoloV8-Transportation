🚗 Vehicle and Pedestrian Detection using YOLOv8
📌 Project Overview

This project focuses on real-time detection of vehicles and pedestrians using the YOLOv8 (You Only Look Once version 8) deep learning model. The system processes images or videos to detect objects and provides information such as object count, detection accuracy, and vehicle speed estimation.

The goal of this project is to build an intelligent traffic monitoring system that can help in traffic analysis, smart city infrastructure, and road safety applications.

🎯 Objectives

.Detect vehicles and pedestrians in real-time.

.Count the number of detected objects.

.Measure vehicle speed using frame-to-frame movement.

.Evaluate model performance using metrics like Precision, Recall, and Accuracy.

.Provide a scalable solution for traffic monitoring systems.
